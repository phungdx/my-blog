# RAG Overview

## What is RAG?

RAG stands for Retrieval-Augmented Generation. It is a technique that combines the power of retrieval and generation to improve the performance of large language models.

## Why RAG?

- LLMs are trained on a massive dataset, but they don't have access to real-time information.
